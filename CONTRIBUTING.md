# 贡献指南

感谢您对 Awesome-Docker 项目感兴趣！我们欢迎并感谢任何形式的贡献。

## 如何贡献

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m '添加一些新功能'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## 贡献内容要求

- 确保添加的资源是高质量的
- 保持分类结构的一致性
- 提供准确的描述和链接
- 遵循现有的格式规范
